# HR Analytics Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes employee attrition, workforce demographics, job satisfaction, and performance trends to help understand the key factors affecting employee retention and organizational performance.

The project focuses on identifying attrition patterns across departments, age groups, salary levels, and job roles through interactive visualizations and business insights.

---

# 🎯 Objectives

- Analyze employee attrition trends
- Identify high-risk departments and employee groups
- Understand the relationship between salary, overtime, and attrition
- Visualize workforce demographics and performance indicators
- Support HR decision-making with interactive dashboards

---

# 🛠 Tools & Technologies Used

- Power BI
- Power Query
- DAX
- Excel/CSV Dataset

---

# 📂 Dataset Information

Dataset: IBM HR Employee Attrition Dataset

The dataset contains:
- Employee demographics
- Salary information
- Job roles and departments
- Attrition status
- Work-life balance
- Overtime data
- Performance and satisfaction metrics

---

# 📊 Dashboard Features

## Executive Overview
- Total Employees
- Attrition Count
- Attrition Rate
- Average Monthly Income

## Attrition Analysis
- Attrition by Department
- Attrition by Age Group
- Attrition by Job Role
- Attrition by Salary Slab

## Employee Insights
- Work-Life Balance Analysis
- Overtime vs Attrition
- Job Satisfaction Analysis
- Gender Distribution

## Interactive Features
- Department Filters
- Job Role Slicers
- Gender Filters
- Dynamic Visual Interactions

---

# 📈 Key Insights

- Employees working overtime showed significantly higher attrition rates.
- Younger employees had comparatively higher attrition.
- Certain departments experienced higher employee turnover.
- Lower salary brackets were more associated with attrition trends.
- Job satisfaction and work-life balance strongly impacted employee retention.
